Introduction to Git and GitHub

Git is a tool for tracking changes in code, allowing developers to collaborate and manage different versions of a project. GitHub is a platform that hosts Git repositories online, enabling easy sharing, collaboration, and version control for developers..

HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines the structure of web pages by using a series of elements like headings, paragraphs, links, images, and more. HTML forms the backbone of most websites and works alongside CSS and JavaScript to create interactive and styled web pages..

CSS

CSS (Cascading Style Sheets) is a language used to style and design the layout of web pages. It controls the appearance of elements written in HTML, such as colors, fonts, spacing, and positioning. CSS allows developers to create visually appealing and responsive websites by separating content from design..

Bootstrap + Tailwind CSS

Bootstrap is a framework with pre-designed components for building responsive websites quickly. Tailwind CSS is a utility-first framework that provides customizable classes to create unique designs with ease.
